INTERACTIVE CALCULATOR
======================

A simple and user-friendly Interactive Calculator built using HTML, CSS, and JavaScript. 
This project performs basic arithmetic operations and updates results in real-time.

----------------------------------------

FEATURES
--------
- Addition, Subtraction, Multiplication, Division
- Real-time calculation
- Responsive and clean UI
- Easy to use interface

----------------------------------------

TECHNOLOGIES USED
-----------------
- HTML (Structure)
- CSS (Styling)
- JavaScript (Logic & Functionality)

----------------------------------------

PROJECT STRUCTURE
-----------------
calculator/
│
├── index.html -> Main HTML file
├── style.css -> Styling file
└── script.js -> JavaScript logic

----------------------------------------

HOW TO RUN
----------
1. Download or clone the repository
2. Open the folder
3. Double click on index.html
4. Use the calculator in your browser
